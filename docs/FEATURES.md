# 맛담 기능 전체 정리

> 🔗 https://matdam-official.vercel.app · 최종 갱신: 2026-07-05

## 🔑 계정
| 기능 | 설명 |
|---|---|
| 아이디 로그인 | 이메일 없이 **아이디 + 비밀번호**로 가입/로그인, 로그인 유지 |
| 프로필 | 닉네임 · 소개 · SNS 링크 · **프로필 사진 업로드** |
| 첫 로그인 가이드 | 처음 로그인 시 사용법 가이드 1회 자동 노출 (마이에서 다시 보기) |

## 📂 내 맛집 리스트
| 기능 | 설명 |
|---|---|
| 개인 리스트 | 나만의 맛집 추가·수정·삭제 |
| 가고싶음 / 방문함 | ❤️ 위시리스트와 ✅ 방문 기록 독립 체크 |
| 우선순위 | 별 1~5개 |
| 사진·태그·메모 | 음식 사진 업로드(또는 URL), 태그, 자유 메모 |
| 검색·필터 | 이름·지역·메모 검색 + **시/도→구 지역 필터** + 카테고리·상태 필터 |

## ✨ 지도 링크 자동 등록
| 기능 | 설명 |
|---|---|
| 링크 자동 인식 | **네이버·구글 지도 공유 링크**만 붙여넣으면 이름·주소·카테고리·대표사진 자동 입력 |
| 메뉴 자동 저장 | 메뉴명·가격·메뉴 사진 최대 20개 함께 저장 |
| 가격대 자동 추정 | 메뉴 가격 중앙값으로 **1인 가격대** 자동 판정 |
| 좌표 자동 저장 | 지도 표시·거리순 정렬용 좌표 저장 |
| 구글 링크 보강 | 구글 링크도 같은 가게를 네이버에서 찾아 동일 수준으로 채움 |

## 🏷️ 맛집 정보
| 기능 | 설명 |
|---|---|
| 카테고리 | 한식 · 한식/고기 · 일식 · **횟집** · 중식 · 양식 · **아시안** · BAR · 카페 · 분식 · 디저트 · 쇼핑 · 생활/문화 (색깔 배지) |
| 💰 가격대 | 만원 이하 / 1~2만원 / 2~4만원 / 4만원 이상 — 카드·상세·피드에 배지 |
| 🍽️ 메뉴판 | 상세 화면에서 메뉴 사진·이름·가격 (6개 초과 시 더 보기) |
| 지도 출처 | N/G 배지 → 탭하면 해당 지도 앱/웹으로 열기 |

## 🌍 전체 맛집 (탐색)
| 기능 | 설명 |
|---|---|
| 통합 피드 | 모든 사용자의 맛집을 **큰 사진 카드** 한 피드로 (같은 맛집은 자동 그룹핑) |
| 정렬 | 🔥 인기순 · ⭐ 별점순 · 👣 방문순 · **📍 가까운순**(위치 권한) |
| 필터 | 지역(시/도→구) · 카테고리 · 맛집/사용자 통합 검색 |
| 🗺️ 맛집 지도 | 전체 맛집을 지도에서 모아보기 — 카테고리색 마커, 팝업(사진·가격대·인기) → 상세, 내 위치 표시 |
| 담기 | 북마크 한 번으로 내 리스트 '가고싶음'에 복사 (메뉴·가격대 포함) |

## 📍 관심 지역 추천
| 기능 | 설명 |
|---|---|
| 관심 지역 설정 | 프로필에 여행지·동네 등록 (쉼표로 여러 곳) — 위치 추적 아님 |
| 홈 추천 피드 | 홈 상단에 그 지역 **추천 맛집 가로 카드** (인기순, 내 리스트 제외) |

## 🧭 테마 컬렉션
| 기능 | 설명 |
|---|---|
| 큐레이션 | "🏞️ 지방 소도시 보석집", "🍜 골목 노포" 등 테마별 맛집 모음 |
| 진입 | 홈 오른쪽 중앙 🧭 버튼 → 컬렉션 목록/상세, 상세에서 바로 담기 |
| 관리 | 관리자가 컬렉션 생성(이모지·제목·소개), 전체 맛집 검색해 추가/제거 |

## 👥 소셜
| 기능 | 설명 |
|---|---|
| 둘러보기 | 사용자 리스트 탐색 — **인기순**(좋아요→조회수→맛집수), ❤️ 리스트 좋아요, 조회수 |
| 리스트 필터 | 다른 사람 리스트도 지역·카테고리로 골라보기 |
| ⭐ 리뷰 | 맛집마다 별점 + 한줄평, 평균 별점 표시 |
| 📊 내 영향력 | 내 맛집을 담아간 사람 수·횟수·인기 맛집 TOP |
| 공유 | 리스트 링크 공유 (웹 복사 / 모바일 공유 시트) |

## 💬 소통 / 관리자
| 기능 | 설명 |
|---|---|
| 피드백 | 유형별(일반·기능요청·버그·정보수정) 전송 |

## 🛠️ 기반 (기술)
| 항목 | 설명 |
|---|---|
| 플랫폼 | 모바일·PC 반응형 웹 (Expo + React Native Web, Vercel 배포) |
| 지도 | OpenStreetMap + Leaflet (API 키·비용 없음) |
| 보안 | 전 테이블 **RLS** — 본인 데이터만 수정, 권한 상승·익명 접근 차단, 스토리지 자기 폴더만 업로드 |
| 성능 | 피드 경량 쿼리 + 60초 캐시 |
| 데이터 | 맛집 316곳 — 카테고리 97% · 가격대 231곳 · 메뉴 240곳 · 좌표 247곳 (자동 백필) |
